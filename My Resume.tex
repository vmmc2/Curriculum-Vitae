\runsubsection{Nemesis}
\descript{| Developer}
\location{Jul. 2019 - Aug. 2019 | Recife - PE, BR}
\begin{tightemize}
\item {Nemesis is a cache simulator of 1024 words that I made using Python. It can simulate 16 different types of cache and supports both write access and read access. It was mainly developed using the RISC-V as a base.}
\item {Relevant technologies used: Python.}
\end{tightemize}

\runsubsection{Hangman Bootloader}
\descript{| Developer}
\location{Mar. 2019 - Apr. 2019| Recife - PE, BR}
\begin{tightemize}
\item {The Hangman Bootloader is as its name suggests a bootloader that can be run in any machine that has Linux as its OS. This bootloader was made using Assembly x86 and contains a hangman game that can be played while the user waits for the system to get initialized.}
\item {Relevant technologies used: Assembly x86; MakeFile.}
\end{tightemize}
